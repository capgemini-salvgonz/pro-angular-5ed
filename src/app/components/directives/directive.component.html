Directives

Descripción

Una directiva en Angular es una característica que nos permite extender la funcionalidad de un elemento HTML existente o crear uno completamente nuevo, agregando ciertas capacidades dinámicas a la página. Las directivas se utilizan para manipular el DOM, añadir atributos personalizados, crear componentes reutilizables y para interactuar con datos, eventos y comportamientos del usuario en una aplicación Angular. La sintaxis básica para definir una directiva en Angular es usando la función `@Directive` y especificando la propiedad `selector` para indicar el elemento al que se aplicará. Por ejemplo: Una vez definida, la directiva puede ser utilizada en el HTML como un atributo, elemento o clase.
