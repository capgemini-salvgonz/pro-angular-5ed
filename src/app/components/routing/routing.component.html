Routing Components

Descripción

El RouterModule agrega directivas y proveedores para la navegación dentro de la aplicación entre las vistas definidas en una aplicación. Angular utiliza el servicio de enrutamiento para especificar de forma declarativa los estados de la aplicación y administrar las transiciones de estado.

Uso

Puede importar este NgModule varias veces, una vez por cada paquete con carga diferida. Sin embargo, solo un servicio de enrutador puede estar activo. Para garantizar esto, hay dos formas de registrar rutas al importar este módulo:

  • El método forRoot() crea un NgModule que contiene todas las directivas, las rutas dadas y el propio servicio de enrutador.
  • El método forChild() crea un NgModule que contiene todas las directivas y las rutas dadas, pero no incluye el servicio de enrutador.
