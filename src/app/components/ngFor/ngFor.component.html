Ng For

Descripción

La directiva ngFor repite una sección de contenido para cada objeto en una matriz, proporcionando la plantilla equivalente a un bucle foreach. El asterisco antes del nombre de la directiva indica que se trata de una directiva de microplantilla.

Las variables más importantes son las que hace referencia al objeto de datos que se está procesando.. Pero la directiva ngFor admite una gama de otros valores que también pueden ser asignados a variables y luego referidos dentro de los elementos HTML anidados, como se describe en la lista siguiente.

Tabla. 1. Valores locales al Template.

Variable

Descripción
index

Este valor numérico es asignado a la posición del objeto actual

count

Este valor numérico es asignado al número de elementos en la fuente de datos
odd

Este es un valor booleano que es puesto en true si el objeto actual se encuentra en una posición impar dentro del data source

even

Este es un valor booleano que es puesto en true si el objeto actual se encuentra en una posición par dentro del data source

first

Este es un valor booleano que es puesto en true si el objeto actual se encuentra al principio del data source

last

Este es un valor booleano que es puesto en true si el objeto actual se encuentra al final del data source

Ejemplo

# Id Producto Precio
{{i+1}} of {{c}} {{product.id}} {{product.name}} {{product.price}}
